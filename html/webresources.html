  5. nodejs
    JavaScript has been mainly used in the front-end field. However, with the advent of 'Node JS', its purpose has completely changed. Thanks to the framework called NodeJS, it is now possible to control server-side technology with JavaScript. It is receiving a lot of attention from web developers, and recently a foundation was established, and technology development is actively taking place. From this website, you can download NodeJS, and it provides information you can learn. NodeJS is used a lot in web applications, so you should check it out if you are interested in it.
